Project Description

Developed a multithreaded banking simulation in Java that replicates banking operations including deposits, withdrawals, transfers, and auditing. Implemented synchronized, cooperating threads to manage multiple accounts concurrently while ensuring data integrity and proper transaction logging. The simulation demonstrates concurrency control, lock management, and inter-thread communication in a realistic scenario.

Skills Learned

  • Multithreading
  • Thread coordination, signaling, blocking
  • Synchronizing shared resources
  • File I/O
  • Debugging and testing
